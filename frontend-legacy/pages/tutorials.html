Biblioteca Técnica (Cards)

📊

FFT & Waterfall

O Waterfall mostra a dissipação de energia. Ressonâncias persistentes aparecem como "caudas" longas. Ideal para tratar acústica de baixa frequência.

🎤

Calibração (94dB)

Use um calibrador de 94dB SPL para ajustar o offset. Carregue o arquivo .cal para compensar a resposta não-linear do seu microfone de medição.

🗺️

Mapa SPL (75-105dB)

A escala ideal para igrejas é de 75dB (fundo) a 105dB (frente). Use o mapa para garantir que o som chegue com a mesma clareza em todos os bancos.

🎚️

Mixer & Auxiliares

Entenda a diferença entre Pré e Pós fader nos monitores. Use o Delay de Aux para alinhar sistemas de reforço lateral ou torres de delay.

🔊

Ruído Rosa & Curva Flat

O ruído rosa tem energia constante por oitava. Use-o para identificar cancelamentos acústicos e buscar uma resposta flat (plana) no sistema de som.

🚫

AFS2 (Anti-Feedback)

O sistema automático deve ser usado para segurança. Cuidado: filtros excessivos degradam a qualidade musical. Ative apenas o necessário para estabilidade.

📱

Mobile & Túnel HTTPS

Navegadores bloqueiam o microfone em conexões não seguras. Use o túnel do SoundMaster para operar via celular com segurança HTTPS total.

🔌

Sincronia com a Mesa

Sempre verifique o status de conexão no rodapé. Se a mesa for reiniciada, use o botão "Reconectar" para restaurar o fluxo de dados em tempo real.

🤖

Assistente IA

O motor IA traduz comandos naturais em ações técnicas na mesa. Peça para "limpar sibilâncias" e ela aplicará filtros De-Esser precisos via regex.

🔄

Alinhamento de Fase

Top e Sub precisam estar em fase. Se o grave "sumir" em certos pontos, inverta a polaridade (Ø) na mesa ou ajuste o delay para somar as ondas.

📈

Análise AVG (Média)

Não equalize baseado em apenas um ponto. Use a função AVG para tirar a média de 4 a 6 pontos da igreja e ter uma leitura fiel da sala.

🎚️

Sends on Faders

Alterne o "Alvo do Mix" para controlar os envios dos músicos diretamente nos faders principais. Prático para ajustes rápidos durante o louvor.

✍️

Nomes Customizados

Renomeie canais e auxiliares para "Voz Pastor" ou "Monitor Piano". O sistema salva automaticamente e persiste mesmo após fechar o app.

📄

Relatórios Acústicos

Exporte seus dados de RT60 e Heatmap para apresentar projetos de tratamento acústico. O SoundMaster gera registros precisos da performance da sala.

📈

Gain Staging

Estrutura de ganho é tudo. Ajuste o Gain para que o sinal fique no verde/amarelo (-18dBFS). Ganho baixo traz ruído; ganho alto traz distorção.

🗜️

Compressão 101

Use o compressor para controlar dinâmicas. Ratio 3:1 para vozes, 4:1 para bumbo/caixa. Ajuste o Threshold para que a redução de ganho seja de 3-6dB.

🎸

Microfonação de Amp

Posicione o mic no centro do cone para mais brilho, ou na borda para um som mais aveludado. A regra de 3:1 ajuda a evitar cancelamentos de fase.

🔔

Corte de Feedback

Identifique a microfonia pelo Analisador. Frequências que "disparam" sozinhas devem ser atenuadas com filtros Notch estreitos no EQ.

👂

Saúde Auditiva

Proteja seus ouvidos e os da igreja. Mantenha a média (Leq) abaixo de 95dB SPL em cultos longos. Use protetores quando estiver próximo ao palco.
